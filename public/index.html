Bad news, due to a hardware failure Santa has lost the Naughty or Nice list.

It seems that all data, the names, the addresses, everything got lost!

Our last hope is in some BASF casettes now, that might contain at least parts of the list. Unfortunately, the elves don't possess the necessary hardware to load it anymore, but somebody out there might be able to help.

We have managed to digitise the contents of the casettes, your puzzle input.

Each row of your input contains a 4ms long section of the audio on the tape, numbers are between -128 and 127. According to the old Datasette manual the casette stores a sequence of 1000, 2000 and 3000Hz audio samples to encode the digits 0, 1 and 2 respectively.

According to the manual the tape contains ASCII text. Each character is encoded separately and uses 8 ternary digits of the input.

The representation is in base 3, for example "10" means decimal 3, "21" is decimal 7.

Each character consists of a leading 0, five data digits and two closing digits 1 and 2 like: 0ddddd12.

The first data digit is the most significant one (MSB order).

Please try to recover whatever you can, and if you like the puzzle let others know about it.
